<task type="auto">
  <name>Task 1: Create Custom Branding App</name>
  <files>apps/escardar_branding/appinfo/info.xml, apps/escardar_branding/style.css</files>
  <action>
    1. Create the directory `apps/escardar_branding`.
    2. Create `appinfo/info.xml` with basic app metadata (name: Escardar Branding, version: 1.0).
    3. Create `style.css` and implement the following CSS variable overrides:
       --color-primary: #2C363F;
       --color-background-plain: #F8F7E5;
       --color-primary-light: #E1E5F2;
       --color-text: #111111;
       --color-background-white: #FFFFFF;
    4. Enable the app using `php occ app:enable escardar_branding`.
  </action>
  <verify>
    <automated>ls apps/escardar_branding/style.css &amp;&amp; php occ app:list | grep "escardar_branding"</automated>
  </verify>
  <done>The custom branding app is created, contains the correct color overrides, and is enabled in Nextcloud.</done>
</task>

<task type="auto">
  <name>Task 2: Configure System Sender Domain</name>
  <files>[]</files>
  <action>
    Use `occ` to configure the system email sender domain to `@escardar.com`.
    Command: `php occ config:system:set mail_domain --value="escardar.com"`
    (And ensure `mail_from_address` is set to a professional address like `noreply@escardar.com`).
  </action>
  <verify>
    <automated>php occ config:system:get mail_domain | grep "escardar.com"</automated>
  </verify>
  <done>System emails are now configured to be sent from the @escardar.com domain.</done>
</task>
